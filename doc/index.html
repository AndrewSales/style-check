libraryhttp://www.andrewsales.com/style-check
Module of BaseX XQuery functions to provide style-checking functionality for Microsoft Word documents. Tested under BaseX 9.5. The target Word XML format is that in the namespace: http://schemas.openxmlformats.org/wordprocessingml/2006/main. The main entry point is sc:check(), which takes one or more paths to *.docx documents. Each of these is unzipped, simplified to consist of elements describing paragraph and inline styles, then validated against the style DTD - see path given in $sc:STYLE_SCHEMA. APPROACH does path exist? is it a DOCX? is it a valid zip? unzip it is it a valid DOCX (does it contain word/document.xml)? transform it to simplified styles validate the result with custom validator return the resulting report TODO: use $sc:UNZIP_DIR
